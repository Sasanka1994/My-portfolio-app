This looks like an "app"!!!

What do you mean by "app" exactly? Regardless, this website looks the way it does because it's using Ionic Framework. It's probably overkill for this use case, and I may change it later, but I'm enjoying Ionic+angular recently.

Also, this is running on NGINX in a Docker container, in AWS' Elastic Container Service!. Why Docker and AWS ECS? Docker is a tool designed to make it easier to create, deploy and run applications by using containers. Thus containers allow to package up an application with all of the part it needs and deploy it as one package. Furthermore AWS' ECS is a fully managed container orchestration service. From this I can make a module to learn CI/CD pipelines in hands on experience.

I am planning to deploy my own automated Jenkins-based build porcess, too; Jenkins will be scanning my remote git repository for changes,then if it finds any, builds and deploys the containerised application automatically, versioning it using per-environment logic and a date-time string.
